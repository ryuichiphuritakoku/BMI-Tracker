### **User Stories**

User Stories เหล่านี้เป็นตัวแทนของความต้องการของผู้ใช้ในมุมมองของฟังก์ชันการทำงานที่สำคัญ โดยเน้นตอบสนองจุดเน้นหลักของโครงการ

| ลำดับ | User Story | จุดเน้นที่เกี่ยวข้อง |
| :---- | :---- | :---- |
| **US1** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ป้อนข้อมูลส่วนสูงและน้ำหนักใหม่**พร้อมระบุวันที่, เพื่อให้**ระบบสามารถคำนวณและบันทึกข้อมูลสุขภาพ ณ วันนั้นได้** | Data Storage & Retrieval |
| **US2** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ให้ระบบตรวจสอบความถูกต้องของข้อมูล (Input Validation)** ก่อนบันทึก, เพื่อให้**มั่นใจได้ว่าฉันป้อนน้ำหนักและส่วนสูงเป็นตัวเลขที่เป็นบวกและสมเหตุสมผล** | Input Validation |
| **US3** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ให้ระบบคำนวณค่า BMI โดยอัตโนมัติ**ทันทีที่ฉันป้อนน้ำหนักและส่วนสูง, เพื่อให้**ฉันทราบดัชนีมวลกายล่าสุดทันที** | Simple Business Logic |
| **US4** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ให้ระบบแสดงผลการคำนวณ BMI** พร้อมกับการจัดกลุ่มตามเกณฑ์มาตรฐาน, เพื่อให้**ฉันเข้าใจสถานะสุขภาพเบื้องต้นของตนเอง** | Basic Reporting |
| **US5** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**เรียกดูประวัติการบันทึกข้อมูลทั้งหมด**ในรูปแบบ**ตาราง**ที่เรียงตามวันที่, เพื่อให้**ฉันสามารถตรวจสอบข้อมูลย้อนหลังได้อย่างรวดเร็ว** | Data Storage & Retrieval, Basic Reporting |
| **US6** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**เห็นกราฟเส้น**ที่แสดงการเปลี่ยนแปลงของค่า BMI ตลอดช่วงเวลาที่บันทึก, เพื่อให้**ฉันเห็นแนวโน้มความคืบหน้าของสุขภาพได้อย่างชัดเจน** | Basic Reporting |
| **US7** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**แก้ไขข้อมูล**ส่วนสูงและน้ำหนักที่บันทึกไว้ในอดีตได้, เพื่อให้**สามารถแก้ไขความผิดพลาดในการป้อนข้อมูลได้อย่างถูกต้อง** | Data Storage & Retrieval |
| **US8** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ลบข้อมูล**การบันทึกสุขภาพรายวันที่ไม่ต้องการได้, เพื่อให้**ข้อมูลประวัติของฉันมีความแม่นยำและเป็นปัจจุบันอยู่เสมอ** | Data Storage & Retrieval |
| **US9** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**ให้ระบบแสดงวันที่และเวลา**ของการบันทึกแต่ละครั้งควบคู่กับค่า BMI, เพื่อให้**ฉันทราบว่าข้อมูล BMI นั้นเป็นข้อมูล ณ วันที่เท่าไหร่** | Data Storage & Retrieval, Basic Reporting |
| **US10** | ในฐานะที่**ผู้ใช้**, ฉันต้องการ**เห็นหน่วยวัด (เช่น kg, m)** กำกับในช่องป้อนข้อมูล, เพื่อให้**ฉันไม่สับสนในการป้อนข้อมูล** | Input Validation |
