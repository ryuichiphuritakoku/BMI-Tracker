**Technology Stack Justification**

1. **ภาษาหลัก: TypeScript (TS)**  
   * คือภาษา JavaScript ที่ "อัปเกรด" ให้เก่งขึ้น (สังเกตจากไฟล์นามสกุล .ts หรือ .tsx)  
   * จุดเด่นคือมีการระบุประเภทข้อมูล (Type) เช่น weight: number ช่วยให้เขียนโค้ดแม่นยำและลดข้อผิดพลาด  
2. **ไลบรารี: React**  
   * เป็นเครื่องมือที่ใช้สร้างหน้าตาเว็บไซต์ (UI)  
   * สังเกตจากการนำเข้า import React ... และการใช้ฟังก์ชันอย่าง useState, useEffect  
3. **รูปแบบการเขียน: TSX (TypeScript XML)**  
   * คือการเขียนโค้ดที่อนุญาตให้เราแทรก **Tag หน้าตาเหมือน HTML** (เช่น \<Router\>, \<div\>) เข้าไปผสมกับโค้ดโปรแกรมได้เลย ทำให้สร้างหน้าเว็บได้สะดวกมาก\\
